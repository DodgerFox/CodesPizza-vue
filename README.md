# codes-vue

## Project setup

1. Create a `.env` file based on `.env.example` and fill in your Firebase credentials.
2. Install dependencies.

```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Preview the production build
```
npm run preview
```

### Lints and fixes files
```
npm run lint
```
